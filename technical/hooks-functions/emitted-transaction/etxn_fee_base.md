### Concepts

* [Emitted Transactions](../../../concepts/emitted-transactions.md)

### Behaviour

* Return the amount of the fee in drops recommended for a to-be emitted transaction.

```c
int64_t etxn_fee_base (
    uint32_t read_ptr,
    uint32_t read_len
);
```
